# zemoso-katas
**Kata 1 - Basic Commits**
```
1) to add files
git add FileName.extension
git status


2) to create new files
touch CreateNewFile

3) to commit message
git commit -m "message"

4) to Overwrite a file
echo contentToAdd > fileName

5)to append a file
echo contentToAdd >> fileName


```

** kata 2 - Staging**
```
git diff
git diff --staged

git restore --staged file.txt


```
